# Golang 全栈开发实战

Golang 全栈开发实战，前后端分离，[书栈网（https://www.bookstack.cn）](https://www.bookstack.cn) 重构实践，项目为 `bookhub`。


## 技术栈

主要技术栈如下：

- Golang
    - fiber
    - xorm
- Vue
    - Nuxt
    - ElementUI
    - uniapp
- 数据库：MySQL
- IDE：VSCODE
